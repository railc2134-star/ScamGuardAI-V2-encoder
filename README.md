# Discord Phishing Scam Detector

This is a Python project that utilizes the Transformers library to detect phishing scams on Discord. It uses a combination of natural language processing (NLP) and machine learning to classify messages as either phishing scams or legitimate messages.

The project uses the following technologies:
- Python 3.x
- PyTorch
- Transformers library
- Discord dataset from Hugging Face

To run this project, follow these steps:
1. Clone the repository
2. Install the required libraries by running `pip install -r requirements.txt`
3. Run the model training script by executing `python transformerSCAM.py`
